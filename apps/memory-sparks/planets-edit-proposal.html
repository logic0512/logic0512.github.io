7 颗星 · 加「编辑」功能 UX 提案 每颗具体加什么、改成什么样

每颗星的录入隐喻不一样，"编辑"也得跟这个隐喻搭得上。
下面 7 颗每颗都列：① 现在录入是什么 ② 编辑加在哪 ③ 点了编辑会发生什么 ④ 隐喻顺不顺
最后我用绿/黄/红三档建议：哪些直接加、哪些有别扭、哪些建议不加。
被子星 悲伤 · 把这件事挂出来晾晒

现在录入是什么

写一段悲伤的话 + 选 1-5 颗水滴权重（越多越湿） → 被子被挂上晾衣绳 → 阳光风吹"晾干"动画 → 文字渐隐

编辑加在哪

历史 modal 的列表项右上角 — 现在已经有删除小垃圾桶，旁边再加一支小铅笔（鼠标 hover 看到）

点了编辑后

1 历史 modal 关闭 → 2 写作 composer 打开，预填上这条原本的文字 + 选中原本的水滴数 → 3 用户改完点「再晾一次」按钮（按钮文案从「挂出来」变成「再晾一次」）→ 4 那床被子重新挂上去 + 走一次晾干动画 → toast「已更新」
✓ 隐喻贴合：被子可以被反复晾，"再晾一次" 比"修改"更对味。推荐做。
花星 爱 · 把这份爱意种下去

现在录入是什么

选花种（玫瑰=爱情 / 康乃馨=亲情 / 茉莉=友情 / 矮牵牛=陪伴 / 天竺葵=关心）+ 写一段话 + 选 1-5 朵权重 → 花圃里长出一朵花

编辑加在哪

点开任意一朵花弹出的 popup 上 — 现有「删除」按钮旁加「修剪」按钮（用花剪图标更对味）

点了编辑后

1 popup 关闭 → 2 composer 打开，预填花种、文字、朵数 → 3 用户可改花种（玫瑰→康乃馨）或文字 → 4 提交后那朵花的品种/颜色直接变化（同位置不挪窝）→ toast
✓ 隐喻贴合：花可以"修剪、重新种"。推荐做。
行星 hike · 带回一张路上的照片

现在录入是什么

上传照片 + 写标题（"山脊尽头的风"）+ 写地点（"梧桐山"）+ 写描述 + 可选给 4 张照片选不同风格滤镜

编辑加在哪

右下角详情卡 — 现在有「删除」按钮和「← →」翻页箭头，加「重写这一段」按钮

点了编辑后

1 详情卡保持不动 → 2 上传 composer 打开，预填地点、标题、描述（照片缩略图也显示，可换可不换）→ 3 提交后这条照片更新（不挪位置、滤镜也保留）→ toast
✓ 隐喻贴合：路边照片回家后整理文字是常态。推荐做。
治愈星 治愈 · 放一颗微光进来

现在录入是什么

写标题（"被认真听完"）+ 描述 + 上传图/语音/视频 + 选 1-5 度温度（越暖光晕越大）

编辑加在哪

点开微光的 reader 详情卡 — 现有「删除」按钮旁加「再写一笔」按钮

点了编辑后

1 reader 关闭 → 2 composer 打开，预填标题、文字、温度（媒体如有则显示原图，可换）→ 3 提交后这颗微光的光晕大小可能变化（温度改了）→ toast
✓ 隐喻贴合：被治愈的瞬间可以反复回想、补充细节。推荐做。
繁星 成就 · 点亮一颗星

现在录入是什么

写成就标题（"完成第一次长线整理"）+ 描述 + 上传图/视频 + 选权重（决定星星大小+亮度）

编辑加在哪

achievementCard 详情卡 — 现有「熄灭这颗星」按钮旁加「擦亮一点」按钮

点了编辑后

1 achievementCard 关闭 → 2 composer 打开，预填标题、描述、权重 → 3 提交后这颗星亮度/大小可能变化（权重改了）→ toast
✓ 隐喻贴合：星星会被你的回顾擦得更亮。推荐做。
树星 孤独 · 写下这段独处

现在录入是什么

写一段独处文字 + 选权重 → 系统自动提取 2-5 个关键词（"凌晨/海边/麻辣烫"）→ 这些关键词变成叶子飘到树上

编辑加在哪

leafCard 详情卡 — 现有「飘落」按钮旁加「再补几笔」按钮

点了编辑后

1 leafCard 关闭 → 2 composer 打开，预填文字 + 权重 → 3 提交后系统重新提取关键词 → 原本的几片叶子淡出，新关键词的叶子重新长出来动画
! 稍有别扭：改文字 → 关键词重算 → 叶子全换。技术能做，但用户改 1 个字可能整片叶子飘走重生，体验会"过激"。
建议：要做，但加一个提示「修改文字会让这条的叶子重新长一遍」。
海星 烦恼 · 写下烦恼，看海水把它带走

现在录入是什么

写烦恼文字 + 选权重 → 文字"写在沙滩上" → 海水冲走动画 → 文字消失，沙滩干净

编辑加在哪

底部 timeline 节点 — 现在已经长按 600ms 触发删除，再加编辑就要做"长按→弹两个选项（重写/删除）"

点了编辑后

1 timeline 节点长按弹小菜单 → 2 点「重写」→ composer 打开，预填文字 + 权重 → 3 提交后那条记录更新 → 重播海水冲走动画
✗ 隐喻冲突：烦恼"已经被海水带走"，回头编辑等于"把已经放下的事捞回来再放下" — 反而打破了"海星帮你忘"的核心。
建议：不加编辑功能。如果用户写错了，可以删了再写新的，更贴合"放下"的隐喻。
