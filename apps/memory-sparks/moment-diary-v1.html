Memory Sparks · 重要时刻 重做

三种「数字日记本」质感,挑一种

PRD 要的是「像一本有质感的深色日记本,私密、文字感强」。下面三个方向用的是 App 里现成的示例时刻,配色字体都跟 App 同一套。每个方向给了时间轴(平时翻看的样子)和展开一篇(沉浸阅读的样子)。情感重量都按 PRD「隐性光晕、不显示数字」处理——你能感觉到哪篇更重,但看不到分数。

右下角可以拉开批注栏边看边记，写完点导出。

方向 A

装帧本

最有「物质感」的一版。深色皮质 / 纸纹的页面，左侧一道装订暗槽像真书的书脊，右上角垂一条缎带书签。正文宋体、首字下沉，日期做成压印小标题。情感重的篇章，纸面会从边缘透出暖金光。翻看时像在翻一本旧日记。

方向 B

素笺

最克制的一版，靠排版和留白取胜，不做拟物。近黑底上一列温润的宋体，大字距、大行距，像一封写给自己的信。日期前一颗小金印，结尾一道发丝细线。情感重量只藏在金印的微光里。最接近 ToWhereOnline 的安静感，也最不会过时。

方向 C

夜光手账

最贴「记忆闪光点」品牌的一版。日记页是一块磨砂玻璃，漂浮在星尘夜空里。情感重的篇章，玻璃页四周会泛起暖金气场——重量直接变成了光。日期用柔蓝（跟愿望色呼应）。跟星球板块是同一个宇宙，氛围最强。
