Memory Sparks · 重要时刻 重做 · 浅色版

三种「浅色日记本」，挑一种

都不要深色，换成明亮的纸页来重做。三个方向用 App 里现成的示例时刻，正文统一宋体。情感重量仍按 PRD「隐性、不显示数字」处理——浅色里改用墨色与暖晕来表达，你能感觉到哪篇更重，但看不到分数。

右下角可拉开批注栏边看边记，写完点导出。

方向 A

暖纸手账

奶油色 / 牛皮纸内页，细微纸纹，顶上一段纸胶带。棕墨色宋体、首字下沉，像 MUJI 笔记本或旅行手账。情感重的篇章，纸面会从底部透出更暖的色。treasure 感最强，像一本会被珍藏的本子。

方向 B

纯白极简

近纯白底，靠排版与留白取胜，不做纸纹。深墨宋体，大字距大行距，日期前一颗小金印，结尾一道发丝细线。像 iA Writer / 苹果备忘录的高级感。冷静、现代、永不过时。

方向 C

米黄信笺

略带米黄的信纸，淡淡的横向信纸细线。宋体正文，桃玫瑰色点缀（呼应 App 里「温柔时刻」的玫瑰色）。情感重的篇章，上缘会晕开一层暖桃色。柔软、亲密、有温度，最像写给在乎的人的信。
