SAND TEXT EFFECT · 沙滩刻字效果对比

同一段文字，三种技术路线的效果呈现

方案一
SVG Filter · 沙粒噪声 + 手工光影合成
feTurbulence + feDisplacementMap + feColorMatrix
质感最真实 无需 JS 参数调试费时

程序生成随机沙粒纹理，字的轮廓被扰动成不规则形状； 分别叠加右下偏移的暗影（字槽深处）和左上偏移的亮光（沙堆受光边缘）， 组合出真实的"刻进去"立体感。

方案二
Canvas · 四层手绘叠加
shadowBlur + fillText × 3 + noise dots
最可控 可加真实纹理图 需要写 JS 字不可选中

依次绘制四层：①字槽深处的暗影（offset 右下+模糊）→ ②主体文字（湿沙深棕）→ ③字槽上沿受光高光（offset 左上）→ ④随机沙粒颗粒噪点。 可以混入真实沙滩照片纹理，上限最高。

把这些烦恼，交给今天的海风。

方案三
纯 CSS · text-shadow 凹陷视觉
text-shadow × 4 + filter: blur
5 行搞定 字可选中 纹理背景上容易穿帮 无沙粒感

用四层 text-shadow 模拟视觉凹陷：左上浅色高光 + 右下暗色阴影 + 扩散柔边 + 轻微 filter:blur 做沙子柔边感。 最省事，但在复杂纹理背景下视觉效果有限。
