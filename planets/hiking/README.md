# 行星原型

这个文件夹收纳行星相册交互相关的独立 HTML 原型。

- `hike-planet-mvp.html`：行星相册主交互页
- `hike-filter-lab.html`：照片滤镜候选页，可上传不同图片对比风格

本地预览：

```text
http://localhost:8017/_prototypes/hike-planet/hike-planet-mvp.html
http://localhost:8017/_prototypes/hike-planet/hike-filter-lab.html
```

根目录下的旧入口仍然保留：

```text
http://localhost:8017/hike-planet-mvp.html
http://localhost:8017/hike-filter-lab.html
```
