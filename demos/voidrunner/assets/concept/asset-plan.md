# Interactive Demo Asset Plan

## Character Lock

Keep these traits fixed across all future generated images:

- Pale white-gray spiky synthetic hair
- Black carbon-metal biomechanical exoskeleton
- Skull-like mechanical face with a small red optic
- Exposed head, neck, and back cables
- Oversized mechanical right forearm cannon with red emitter glow
- Long clawed left hand
- Hunched predator posture, lean and asymmetrical body
- Restrained monochrome palette with small red accents

Avoid existing IP, logos, readable text, human skin, colorful superhero styling, fantasy horns/wings, duplicated limbs, and baked-in UI.

## Current Assets

- `character-master.png`: approved master character direction.
- `menu-hero.png`: white studio menu hero with left negative space for HTML/CSS menu.
- `customization-hero.png`: standing character render for equipment/customization screen.
- `match-transition.png`: white-space particle dissolve frame for Start Match transition.
- `city-environment.png`: rainy cyberpunk street background without hero character, good for parallax.
- `gameplay-hero.png`: third-person gameplay frame with character from behind.
- `asset-sheet.jpg`: quick visual contact sheet of the current batch.

## Suggested Demo Flow

1. Main menu: use `menu-hero.png`; layer narrow menu text and hover selection on the left.
2. Customization: use `customization-hero.png`; build UI in CSS with left category nav, item list, and stats panel.
3. Match start: fade UI out, scale character slightly, overlay particle canvas; transition to `match-transition.png`.
4. Environment load: crossfade white void into `city-environment.png` with fog mask.
5. Gameplay: use `gameplay-hero.png`; add HUD in CSS/canvas and animate red cannon glow or targeting line.

## Reusable Prompt Core

Same original bio-mechanical humanoid hunter archetype: black carbon-metal exoskeleton, pale white-gray spiky synthetic hair, skull-like mechanical face with one small red optical lens, exposed cables from head to back, layered armor plates, long clawed hands, oversized mechanical right forearm cannon with red emitter glow, hunched predator posture, lean asymmetrical body, premium AAA sci-fi game concept art, photoreal 3D render, restrained monochrome palette with subtle red accents.
