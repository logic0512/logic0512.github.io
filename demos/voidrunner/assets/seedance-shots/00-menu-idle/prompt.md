# 00 Menu Idle

## Reference Images

- `menu-hero-reference.png`

## Goal

Optional main-menu idle clip. The character stays crouched, but feels alive before the user clicks Start.

## Seedance Prompt

Animate this image as a premium AAA sci-fi game main-menu idle shot. Keep the same crouched bio-mechanical hunter in a clean pale gray-white studio space. The character barely moves: slow mechanical breathing, slight shoulder rise and fall, tiny head tilt, subtle cable sway, faint red glow pulsing in the optic and right forearm cannon. Keep the background stable and minimal. No UI, no menu text, no logo, no extra characters.

## Settings

- Duration: 3-4 seconds
- Camera: locked or extremely slow push-in
- Output: `menu-idle.mp4`

## Avoid

Do not make the character stand up. Do not change the pose dramatically. Do not add interface elements.
