# 10 Match Default Cannon

## Reference Images

- `start-default-cannon.png`
- `dissolve-style.png`
- `end-gameplay.png`

## Goal

Generate the match-entry animation for the default kinetic cannon. This replaces the shared generic entry clip when the default weapon is equipped.

## Seedance Prompt

Use the first image as the starting weapon state, the second image as the dissolve / particle style reference, and the third image as the final gameplay environment. Animate the same black biomechanical hunter with white spiky hair and the default heavy forearm cannon entering the match. Start in the pale gray-white customization space. The character leans forward, the default cannon red emitter pulses, the body edges and cables break into dark particles and mist, then the character is pulled into a rainy cyberpunk industrial street. End as a third-person gameplay view from behind, keeping the default cannon silhouette readable and the red muzzle glow visible. Premium AAA sci-fi game cinematic, heavy mechanical motion, no HUD, no UI, no readable text, no logos.

## Settings

- Duration: 4-5 seconds
- Output: `match-default.mp4`
