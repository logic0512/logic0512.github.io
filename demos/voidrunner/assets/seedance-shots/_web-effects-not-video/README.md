# Web Effects: Do Not Generate In Seedance

These effects will be created directly in the webpage, because they need to respond to clicks, timing, layout, and browser state:

- Menu hover highlight
- Button glow
- Equipment card selection
- Stat bar animation
- HUD overlay
- Crosshair
- Scan lines
- Corner UI panels
- Cursor interactions
- Page fade / wipe transitions
- Video masks and compositing

If these are generated inside the video, they will be harder to align and less interactive.
