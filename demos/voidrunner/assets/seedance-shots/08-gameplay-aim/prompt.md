# 08 Gameplay Aim

## Reference Images

- `gameplay-reference.png`

## Goal

Optional gameplay action clip. The character moves from idle into an aiming / ready stance. HUD and targeting UI will be added later in the webpage, so do not generate UI here.

## Seedance Prompt

Animate this image as a third-person AAA sci-fi gameplay moment in a rainy cyberpunk industrial street. The bio-mechanical hunter is seen from behind. The character shifts weight, raises the right forearm weapon slightly, shoulders tense, exposed cables sway, the red weapon emitter brightens, rain and steam move through the environment, wet pavement reflections shimmer. Keep the motion restrained and game-like, as if the player is preparing to aim. No HUD, no crosshair, no UI, no readable text, no logos.

## Settings

- Duration: 2-3 seconds
- Camera: subtle third-person drift, no fast pan
- Output: `gameplay-aim.mp4`

## Avoid

Do not fire a huge blast. Do not add enemies, explosions, or interface elements.
